Paso a Paso para Ejecutar la Solución

1. Clonar el repositorio:

git clone <repository-url>
cd <repository-directory>

2. Configurar el Backend:

cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload

3. Configurar el Frontend:

cd frontend
npm install
npm start

4. Despliegue con CloudFormation:

Crear un archivo cloudformation.yaml con la configuración necesaria.

Ejecutar el despliegue:
aws cloudformation create-stack --stack-name my-stack --template-body file://cloudformation.yaml
